5 Algorithm block

We consider the simple algorithm description philosophy appeared in Jeff Erickson’s jeffe.sty.

EuclidianGCD(a,b)
  while a\neq b
      if a>b
          a\gets a-b
      else
          b\gets b-a
  return a
